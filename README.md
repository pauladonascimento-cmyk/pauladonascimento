# Site Paula do Nascimento

Site institucional premium para GitHub Pages.

## Arquivos

- `index.html` — conteúdo e estrutura do site.
- `style.css` — identidade visual, responsividade e layout.
- `script.js` — menu mobile, animações e formulário para WhatsApp.
- `assets/` — fotografias originais e favicon.

## Observação sobre as fotos

As fotografias foram mantidas como arquivos originais. O site faz apenas recorte visual por CSS com `object-fit`, sem alterar características físicas, rosto, pele ou aparência.

## Publicação no GitHub Pages

1. Entre no repositório `pauladonascimento.github.io`.
2. Faça upload de todos os arquivos e da pasta `assets`.
3. Clique em `Commit changes`.
4. Aguarde alguns minutos.
5. Acesse: `https://pauladonascimento.github.io`.
